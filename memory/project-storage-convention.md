所有代码开发项目统一放到 `E:\.Claude Code Project\` 路径下。

**新建项目规则**：
1. 在该路径下新建项目文件夹
2. 所有项目产出物保存在该文件夹内
3. 文件夹命名规则：`序号.项目名称_创建日期`
   - 序号：根据已有项目文件夹序号向下顺排（如已有 1 和 2，则新项目为 3）
   - 创建日期：8 位数字 YYYYMMDD

**示例**：
- `E:\.Claude Code Project\1.新闻网站爬虫_20260501\`
- `E:\.Claude Code Project\2.版本管理系统_20250501\`

**Why**：用户要求统一项目存储位置，便于管理和查找。

**How to apply**：每次创建新项目时，先列出 `E:\.Claude Code Project\` 下已有文件夹确定序号，再按规则创建。
